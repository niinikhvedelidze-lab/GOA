Cascading Style Sheets

**CSS (Cascading Style Sheets)** is a stylesheet language used to control the **appearance and layout** of HTML elements on a webpage. It allows developers to apply styles like **colors, fonts, spacing, and positioning** to make websites visually appealing. CSS can be added inline, internally within an HTML file, or externally through separate `.css` files. Modern CSS includes features like **Flexbox**, **Grid**, **animations**, and **variables**, enabling responsive and dynamic web designs that adapt to different screen sizes and devices.
