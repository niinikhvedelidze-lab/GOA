What is Cyber Security?

Cyber Security is the practice of protecting systems, networks, and data from digital attacks. It involves several areas including ethical hacking, forensics, and network defense.

Here are some key areas in cyber security:

  1. Network Security
  2. Application Security
  3. Information Security
  4. Incident Response
  5. Ethical Hacking

Introduction to Operating Systems

Operating systems are the foundation of all computer activities. In cyber security, it’s important to understand how each one works.

  • Windows – The most common OS; often targeted by attackers.
  • Linux (Kali Linux) – Popular for ethical hacking and penetration testing.
  • macOS – Known for stability and security features.

Kali Linux – A specialized Linux system for security testing.
